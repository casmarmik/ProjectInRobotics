7.1.12 RSI_DISABLE()

Description

RSI_DISABLE() désactive un conteneur RSI.

Un conteneur désactivé doit être réactivé avant d'activer le traitement du signal avec RSI_ON(). RSI_ENABLE() active un conteneur désactivé.

Syntaxe

RET=RSI_DISABLE(ID conteneur : IN)

Explication de la syntaxe

Elément

Description

RET

Type : INT

Valeurs retournées :

■

RSIOK : fonction exécutée avec succès

■

RSIINVOBJID : ID de conteneur invalide

ID conteneur : IN

Type : INT

ID du conteneur RSI
