4.5 Désinstallation de RSI Visual

Condition préalable

■

Droits d'administrateurs locaux

Procédure

1.

Dans le menu de démarrage Windows, sous Paramètres > Panneau de configuration > Logiciel, effacer l'entrée RSIVisual.

2.

Effacer le dossier RSIVisual dans le répertoire C:\Programmes\KUKA Roboter GmbH.
