7.1.12 RSI_DISABLE()

Beschreibung

RSI_DISABLE() deaktiviert einen RSI Container.

Ein deaktivierter Container muss vor dem Einschalten der Signalverarbeitung mit RSI_ON() wieder aktiviert werden. RSI_ENABLE() aktiviert einen deaktivierten Container.

Syntax

RET=RSI_DISABLE(ContainerID:IN)

Erläuterung der Syntax

Element

Beschreibung

RET

Typ: INT

Rückgabewerte:

■

RSIOK: Funktion erfolgreich ausgeführt

■

RSIINVOBJID: Ungültige Container-ID

ContainerID:IN

Typ: INT

ID des RSI Containers
