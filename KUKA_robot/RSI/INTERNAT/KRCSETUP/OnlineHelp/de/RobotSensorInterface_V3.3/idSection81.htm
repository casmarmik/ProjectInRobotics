11.2.8 RSI Objekte zur Signalsteuerung

Name

Beschreibung

P

Verstärkung des Signals

PD

Proportional-Differential-Objekt

y(k) = B0 * x(k) + B1 *x(k-1)

B0 = KR * (1 + (TV / <Sensortakt>))

B1 = -KR * (TV / <Sensortakt>)

I

Integrationsobjekt (Trapez-Algorithmus)

y(k) = B0 * (x(k) + x(k-1)) + y(k-1)

B0 = <Sensortakt> / (2 * TI)

D

Differenzierungsobjekt

y(k) = B0 * (x(k) - x(k-1))

B0 = KD / <Sensortakt>

PID

PID-Objekt

y(k) = y(k-1) + B0 * x(k) + B1 * x(k-1) + B2 * x(k-2)

B0 = KR * (1 + TV / <Sensortakt>)

B1 = - KR * (1 - <Sensortakt> / TN + 2 * TV / <Sensortakt>)

B2 = KR * TV / <Sensortakt>

PT1

Verzögerungsobjekt 1. Ordnung

y(k) = - A0 * y(k-1) + B0 * x(k)

A0 = -exp(-<Sensortakt> / T1)

B0 = KR * (1 - exp(-<Sensortakt> / T1)

PT2

Verzögerungsobjekt 2. Ordnung

y(k) = - A0 * y(k-1) - A1 * y(k-2) + B0 * x(k) + B1 *x(k-1)

Fall 1: T1 != T2

■

Z1 = exp(-<Sensortakt> / T1)

■

Z2 = exp(-<Sensortakt> / T2)

■

A0 = -Z1 - Z2 A1 = Z1 * Z2

■

B0 = (KP / (T1 - T2)) / (T1 * (1 - Z1) - T2 * (1 - Z2))

■

B1 = (KP / (T1 - T2)) / (T2 * Z1 *(1 - Z2) - T1 * Z2 *(1 - Z1))

Fall 2: T1 == T2

■

Z0 = exp(-<Sensortakt> / T1)

■

B0 = KP * (1 - Z0 * ((<Sensortakt> / T1) + 1))

■

B1 = KP * Z0 * (Z0 + (<Sensortakt> / T1) - 1)

GENCTRL

Generisches Signalverarbeitungsobjekt bis zur 8. Ordnung

y(z) = B0*u(z) + B1*u(z-1) + B2*u(z-2) +...+ B8*u(z-8) - A1*y(z-1) - A2*y(z-2) -...- A8*y(z-8)

IIRFILTER

IIR-FILTER
