6.1.3 Inserting and linking a comment

Procedure

1.

Drag a comment object into the editor.

2.

Select the text box and enter the comment.

3.

Select the Comment Connector tool in the Toolbox.

4.

Point to the comment with the mouse pointer.

5.

Once the link icon is displayed on the comment, click on it and point to the desired RSI object with the mouse pointer.

6.

Once the link icon is displayed on the RSI object, click on it again.
