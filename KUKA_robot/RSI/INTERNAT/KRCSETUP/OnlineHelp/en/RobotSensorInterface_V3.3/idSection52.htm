7.3.2 Modifying the signal flow parameters in KRL

Description

Signal flow parameters can be modified subsequently by means of the following functions in the KRL program.

■

RSI_GETPUBLICPAR(): reads the configured value of an RSI object parameter.

■

RSI_SETPUBLICPAR(): assigns a new value to the RSI object parameter.

Precondition

■

RSI object parameter is enabled.
