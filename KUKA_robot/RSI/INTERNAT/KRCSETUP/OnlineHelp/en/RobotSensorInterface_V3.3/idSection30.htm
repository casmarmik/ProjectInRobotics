The scaling of the ordinate is automatically based on the largest signal assigned to it.

5

The check box must be activated to display a signal in the monitor.

Default: Check box for Signal 1 … 6 activated.

The following buttons are available:

Button

Description

Activate all

Activates all signals.

Deactivate all

Deactivates all signals.

Reset thickness

Resets the line thicknesses of the signals to the preset line thickness 1.

Reset colors

Resets the signal colors to the preset colors.

Reset monitor

Resets the time window to the preset size (cancel zoom).

Reset all?

Resets all signal properties to the preset values.
