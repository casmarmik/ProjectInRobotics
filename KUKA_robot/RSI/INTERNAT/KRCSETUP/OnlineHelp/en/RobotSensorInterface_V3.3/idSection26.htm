6.1.5 Enabling an RSI object parameter

Description

It is possible to read the value of an RSI object parameter in the KRL program and subsequently assign a new value to the object parameter.

(>>> 7.3.2 "Modifying the signal flow parameters in KRL")

A precondition for this is that the parameter has been enabled in the signal flow configuration.

Procedure

1.

Select an RSI object parameter in the signal flow editor.

The properties of the parameter are displayed in the Properties window.

2.

Set the IsPublic box to True.

Setting parameters for specific objects during the runtime of the RSI is not permissible. In such a case, the ‘IsPublic’ box cannot be set to ‘True’.
