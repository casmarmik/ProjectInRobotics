11.2.9 Other RSI objects

Name

Description

TIMER

On expiry of the set time, a positive edge is set on signal output “Out1”.

LIMIT

Limits the signal to values within a lower and upper limit (LowerLimit, UpperLimit).

MINMAX

Returns the current smallest and largest signal across all input signals.

Up to 10 input signals can be connected.

DELAY

Delays the input signal by a defined time.

SIGNALSWITCH

Switches between 2 signal paths via control signal.

ETHERNET

UDP Ethernet communication in XML data format

Up to 64 inputs and outputs can be defined in the configuration file. Signals at the inputs are sent to the communication partner. The data received from the communication partner are available at the outputs.
